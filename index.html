Sublime-AutoSetNewFileSyntax

This plugin automatically sets the syntax for a plain text content depends on its first line. The original thought is from here.

Examples

If we create a new file in Sublime Text and type <?php, this plugin will set the syntax to PHP automatically.

How It Works

  1. Read all syntax definition files.
  2. Try to find first_line_match in .sublime-syntaxs and firstLineMatch in .tmLanguages.
  3. Match the first line with results we found in the last step.
  4. If there is any luck, set the corresponding syntax for the user.

Future Works

  • Let user add their own first_line_match and the corresponding syntax pairs.
